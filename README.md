# Password Manager

This is a simple command-line password manager written in Java. It allows users to generate, store, display, and search for passwords associated with different websites and email addresses.

## Features

- **Password Generation**: Users can generate strong random passwords of specified length.
- **Storage**: Passwords are stored in a CSV file (`passwords.csv`) for persistence across sessions.
- **Display**: Users can display all stored passwords along with their associated website and email.
- **Search**: Users can search for a password by providing the website name.

## Setup

1. Clone the repository to your local machine.
2. Compile the Java file: `javac pass.java`
3. Run the compiled Java program: `java pass`

## Usage

Upon running the program, users are presented with a menu:

1. **Generate a new password**: Generate a new password for a website and email address.
2. **Display all passwords**: Display all stored passwords along with their associated website and email.
3. **Search for a password**: Search for a password associated with a specific website.
4. **Exit**: Exit the program.

Follow the prompts to perform the desired action.

## Dependencies

- Java Development Kit (JDK)
